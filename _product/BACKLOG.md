# Backlog

This is your inbox. Capture everything here — no formatting required.
When an entry is ready to build, use Claude to turn it into a Brief.

Types: `feature` / `bug` / `ux` / `debt` / `experiment`
Status tags: `#inbox` `#ready` `#in-progress` `#done` `#dropped`

---

## How to capture

Add a new entry at the top. Two lines max. Don't overthink it.

```
## [date] [type] #inbox
[what you want, why it matters]
```

---

## How to promote to a Brief

Paste the entry into Claude with this prompt:

```
Turn this backlog entry into a complete implementation brief using my brief template.
Entry: [paste entry]
```

Save the output to _product/BRIEFS/[number]-[slug].md and mark this entry #ready.
